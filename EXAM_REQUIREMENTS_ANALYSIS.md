### 2. 考试计时功能

**功能要求**:
- 90分钟倒计时
- 实时显示剩余时间
- 时间到自动交卷
- 暂停/继续功能（可选）
- 时间不足5分钟提醒

**实现方式**:
```javascript
// 前端倒计时
const TIME_LIMIT = 90 * 60; // 90分钟（秒）
let remainingTime = TIME_LIMIT;

const timer = setInterval(() => {
  remainingTime--;
  updateTimerDisplay();

  if (remainingTime <= 300) { // 5分钟
    showTimeWarning();
  }

  if (remainingTime <= 0) {
    clearInterval(timer);
    autoSubmit();
  }
}, 1000);
```

## 💡 技术要点

### 组卷算法关键点

1. **按比例分配**: 确保各分类题目数符合考试要求
2. **随机抽取**: 每次考试题目不同
3. **去重**: 同一题目不重复出现
4. **备用题库**: 如果某分类题目不足，从其他分类补充

### 计时功能关键点

1. **前端计时**: 使用setInterval实现倒计时
2. **时间同步**: 记录开始时间到后端
3. **异常处理**: 页面刷新后恢复计时状态
4. **自动交卷**: 时间到自动提交答案
