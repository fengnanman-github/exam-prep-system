# 前端配置管理界面 - 完成报告

## 功能概述

已成功开发前端配置管理界面，管理员可以通过可视化的Web界面动态调整系统的题目范围、文档范围和自动更新规则，无需重启系统即可生效。

## 界面结构

### 主组件：AdminConfig.vue

**位置**: `frontend/src/components/AdminConfig.vue`

**标签页**:
1. 📝 题目范围配置
2. 📚 文档范围配置
3. 🤖 自动规则
4. 📜 配置历史

### 子组件

#### 1. QuestionScopeConfig.vue
**功能**: 配置5种练习模式的题目范围

**支持的模式**:
- **全部题目** (all): 使用所有题目
- **按知识点分类** (category): 通过多选框选择知识点
- **按考试类别** (exam_category): 通过多选框选择考试类别
- **自定义题目** (custom): 手动输入题目ID列表
- **按关联文档** (document): 手动输入文档ID列表

**通用功能**:
- 排除ID功能（任何模式下都可用）
- 实时预览选中的题目/文档数量
- 重置和保存操作

#### 2. DocumentScopeConfig.vue
**功能**: 配置2种文档模式的文档范围

**支持的模式**:
- **全部文档** (all): 使用所有文档
- **按分类** (category): 通过多选框选择文档分类
- **自定义文档** (custom): 手动输入文档ID列表

**功能**:
- 排除ID功能
- 实时预览选中的文档数量
- 重置和保存操作

#### 3. AutoRulesConfig.vue
**功能**: 控制系统的自动更新行为

**规则**:
- **自动应用SuperMemo算法**: 练习后自动计算复习时间
- **自动同步题目状态**: 跨模式同步不确定标记和收藏
- **自动更新掌握度**: 根据练习结果更新掌握等级

**交互**:
- 优雅的开关按钮
- 实时状态显示
- 即时生效

#### 4. ConfigHistory.vue
**功能**: 查看配置变更历史

**显示信息**:
- 配置名称
- 变更时间
- 操作者
- 变更原因
- 变更前后值对比

## API集成

### 后端API调用

**获取配置**:
- `GET /api/v2/admin/question-scopes` - 获取题目范围配置
- `GET /api/v2/admin/document-scopes` - 获取文档范围配置
- `GET /api/v2/admin/system-config` - 获取所有系统配置

**更新配置**:
- `PUT /api/v2/admin/question-scope/:type` - 更新题目范围
- `PUT /api/v2/admin/document-scope/:type` - 更新文档范围
- `PUT /api/v2/admin/system-config/:key` - 更新系统配置

**历史记录**:
- `GET /api/v2/admin/system-config/history` - 获取配置历史

**元数据**:
- `GET /api/v2/admin/categories` - 获取知识点分类
- `GET /api/v2/questions/exam-categories` - 获取考试类别

### 认证
- 所有API调用使用JWT token认证
- Token从localStorage获取
- 自动处理认证错误

## 技术栈

- **Vue 3**: 组合式API (Composition API)
- **响应式系统**: ref, watch, computed
- **组件通信**: props, emits
- **HTTP请求**: fetch API
- **状态管理**: localStorage token

## 访问方式

1. 登录系统（使用管理员账号）
2. 点击导航栏的"⚙️ 系统配置"按钮
3. 选择相应的标签页进行配置
4. 修改配置后点击"保存配置"按钮
5. 配置立即生效，无需刷新页面
