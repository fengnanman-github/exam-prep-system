### 第一步：书名号提取（精确标注）

**原理**: 从题干中提取《》内的文档名称进行精确匹配

**执行结果**:

- 🎯 精确度: 99%+（几乎无误匹配）

**示例**:
```

```

### 第二步：关键词补充（覆盖剩余）

**原理**: 通过预定义关键词在未标注题目中进行模糊匹配

**执行结果**:
- ✅ 又标注: **3,055 道题目**
- 📊 补充率: 60.19%
- ⚠️ 精确度: 约85%（可能有少量误匹配）

**示例**:
```

映射: 网络安全法 ✅
```

## 📊 数据导出

### 导出标注结果

```sql
-- 导出完整的标注数据
COPY (
  SELECT
    id,
    question_no,
    question_text,
    original_document,
    document_category,
    document_priority
  FROM questions
  WHERE original_document IS NOT NULL
  ORDER BY document_priority DESC, original_document
) TO '/tmp/labeled_questions.csv' WITH CSV HEADER;
```

### 导出未标注题目

```sql
-- 导出未标注的题目（待人工处理）
COPY (
  SELECT
    id,
    question_no,
    question_text
  FROM questions
  WHERE original_document IS NULL
  ORDER BY id

```
