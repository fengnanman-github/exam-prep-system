## 功能概述

系统配置管理功能允许您动态控制：

| 功能 | 说明 |
|------|------|
| 📝 题目范围配置 | 控制各种练习模式使用的题目池 |
| 📚 文档范围配置 | 控制练习和文档复习模式使用的文档库 |
| 🤖 自动规则 | 控制SuperMemo算法、状态同步等自动功能 |
| 📜 配置历史 | 查看所有配置的变更记录 |

**核心特性**：
- ✅ 实时生效，无需重启系统
- ✅ 配置永久保存，重启后自动恢复
- ✅ 支持配置回滚和审计
- ✅ 灵活的范围控制（全部、分类、自定义等）

## 一、题目范围配置

### 概述

题目范围配置用于控制**5种练习模式**使用的题目池：

| 配置项 | 对应的练习模式 | 作用 |
|--------|---------------|------|
| 🎲 随机练习 | 随机抽题 | 控制智能推荐、随机抽题的题目池 |
| 📂 分类练习 | 按知识点分类练习 | 控制按知识点分类时的题目范围 |
| 🎯 考试类别 | 按考试类别练习 | 控制按考试类别时的题目范围 |
| 📄 文档复习 | 文档关联题目 | 控制文档复习模式关联的题目 |
| 🎲 随机抽题 | 随机练习模式 | 控制随机练习模式的题目范围 |

### 配置界面说明

每个配置项包含以下内容：

```
┌─────────────────────────────────────────────────┐
│ 🎲 随机练习                    [全部题目] │ ← 范围模式
├─────────────────────────────────────────────────┤
│                                         │
│ 范围模式: [▼全部题目            ]        │ ← 模式选择下拉框
│                                         │
│ ┌───────────────────────────────────────┐   │
│ │ ☑ 密码管理                       │   │ ← 分类选择框（按分类模式时显示）
│ │ ☑ 密码算法                       │   │
│ │ ☑ 密码产品                       │   │
│ │ ☑ PKI体系                        │   │
│ │ ☑ 密码协议                       │   │
│ └───────────────────────────────────────┘   │
│                                         │
│ 题目ID列表 (用逗号分隔):              │ ← 自定义输入框（自定义模式时显示）
│ ┌───────────────────────────────────────┐   │
│ │ 1, 2, 3, 5, 8, 13...            │   │
│ └───────────────────────────────────────┘   │
│ 当前选中 10 个题目                      │
│                                         │
│ 排除题目ID (用逗号分隔):              │ ← 排除输入框（任何模式都显示）
│ ┌───────────────────────────────────────┐   │
│ │ 1, 2, 3, 4, 5                    │   │
│ └───────────────────────────────────────┘   │
│ 当前排除 5 个题目                      │
│                                         │
│ [重置]  [保存配置]                     │ ← 操作按钮
└─────────────────────────────────────────────────┘
```

### 范围模式详解

#### 模式1：全部题目 🔵

**说明**：使用题库中所有题目（5075题）

**适用场景**：
- 默认配置
- 需要使用完整题库进行练习

**操作步骤**：
1. 点击"范围模式"下拉框
2. 选择"全部题目"
3. 点击"保存配置"

---

#### 模式2：按知识点分类 🟢

**说明**：只使用指定知识点分类的题目

**可用分类**（可多选）：
- ☑ 密码管理
- ☑ 密码算法
- ☑ 密码产品
- ☑ PKI体系
- ☑ 密码协议

**适用场景**：
- 专项练习某个知识点
- 针对薄弱环节进行强化

**操作步骤**：
1. 点击"范围模式"下拉框
2. 选择"按知识点分类"
3. 勾选需要的知识点分类（可多选）
4. 点击"保存配置"

**示例配置** - 专项练习密码算法：
```
范围模式: 按知识点分类

选择知识点分类:
☑ 密码算法
☐ 密码管理
☐ 密码产品
☐ PKI体系
☐ 密码协议
```

**说明**：只使用指定考试类别的题目

**可用类别**（可多选）：

#### 模式4：自定义题目 🟩

**说明**：只使用指定ID列表的题目

**适用场景**：
- 练习错题集
- 练习重点题

1. 点击"范围模式"下拉框
2. 选择"自定义题目"
3. 在文本框中输入题目ID，用逗号分隔

**说明**：只使用与指定文档关联的题目

**适用场景**：
- 基于特定法规/标准进行练习
- 文档题目联动练习


### 排除功能详解

**说明**：在任何范围模式下，都可以排除特定题目

**适用场景**：
- 排除已知有问题的题目
- 排除已完全掌握的题目
- 临时排除某些题目用于测试

**操作步骤**：
1. 找到"排除题目ID"文本框
2. 输入要排除的题目ID，用逗号分隔
3. 点击"保存配置"

**示例配置** - 排除已知问题题目：
```
范围模式: 全部题目

排除题目ID (用逗号分隔):
1, 2, 3, 4, 5, 100, 101, 102

当前排除 8 个题目
```

**效果**：
- 从题目池中移除这8道题目
- 其他所有题目（5075 - 8 = 5067题）仍然可用

---

### 保存与重置

**保存配置**：
1. 修改配置后，点击"保存配置"按钮
2. 配置立即生效，无需重启系统
3. 系统会弹出"配置已更新"提示

**重置配置**：
1. 点击"重置"按钮
2. 恢复到**上次保存**的配置状态
3. 不会清除已保存的配置

**提示**：重置不会将配置恢复到默认值，而是撤销本次的修改。

## 三、自动规则配置

### 概述

自动规则配置用于控制系统是否自动应用各种算法和同步功能。

### 规则列表

```
┌─────────────────────────────────────────────────────┐
│ 自动应用SuperMemo算法                [●] 已启用 │
│ 练习提交后自动应用SuperMemo间隔重复算法...      │
├─────────────────────────────────────────────────────┤
│ 自动同步题目状态                 [○] 已禁用 │
│ 跨模式自动同步题目状态（不确定标记、收藏等）   │
├─────────────────────────────────────────────────────┤
│ 自动更新掌握度                   [●] 已启用 │
│ 根据练习结果自动更新题目的掌握等级          │
├─────────────────────────────────────────────────────┤
│                                             │
│ ℹ️ 规则说明                               │
│ ┌───────────────────────────────────────────┐    │
│ │ • 自动应用SuperMemo算法              │    │
│ │   练习后自动计算下次复习时间         │    │
│ │                                        │    │
│ │ • 自动同步题目状态                  │    │
│ │   跨模式同步不确定标记和收藏状态     │    │
│ │                                        │    │
│ │ • 自动更新掌握度                    │    │
│ │   根据练习结果自动更新题目掌握等级   │    │
│ └───────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
```

### 规则详解

#### 规则1：自动应用SuperMemo算法

**功能**：使用SuperMemo间隔重复算法计算复习时间

**工作原理**：
- ✅ 答对题目 → 增加复习间隔（如1天 → 3天 → 7天）
- ❌ 答错题目 → 重置复习间隔（立即需要复习）
- 根据答题质量调整难度系数

**启用方法**：
1. 找到"自动应用SuperMemo算法"规则
2. 点击开关，滑块移到右侧
3. 状态变为"已启用"（绿色）

**推荐配置**：🔵 **启用**
- 这是科学的复习算法，能显著提高学习效率
- 帮助避免遗忘和过度复习

---

#### 规则2：自动同步题目状态

**功能**：跨模式自动同步题目状态

**同步内容**：
- ❓ 不确定标记
- ⭐ 收藏标记

**工作原理**：
- 在"随机练习"中标记某题为不确定
- 切换到"分类练习"时，该题的不确定标记仍然可见

**启用方法**：
1. 找到"自动同步题目状态"规则
2. 点击开关启用

**推荐配置**：🔵 **启用**
- 提供一致的用户体验
- 避免在不同模式下重复标记

---

#### 规则3：自动更新掌握度

**功能**：根据练习结果自动更新题目掌握等级

**掌握等级**：
- 🟢 未开始 → 从未练习过
- 🟡 学习中 → 刚刚开始练习
- 🟠 挣扎中 → 经常答错，需要加强
- 🟢 已掌握 → 正确率较高，基本掌握

**工作原理**：
- 根据最近N次答题的准确率计算
- 答对次数多 → 掌握度上升
- 答错次数多 → 掌握度下降

**启用方法**：
1. 找到"自动更新掌握度"规则
2. 点击开关启用

**推荐配置**：🔵 **启用**
- 跟踪学习进度
- 帮助识别薄弱环节

## 四、配置历史

### 功能说明

查看所有配置的变更历史记录。

### 历史记录内容

每条记录包含：

| 字段 | 说明 |
|------|------|
| 配置项 | 变更的配置名称 |
| 变更时间 | 变更发生的时间 |
| 变更人 | 执行变更的管理员用户名 |
| 旧值 | 变更前的值 |
| 新值 | 变更后的值 |
| 变更原因 | 可选的变更说明 |

### 如何查看

1. 点击"配置历史"标签页
2. 查看历史记录列表（按时间倒序，最新的在前）

### 使用场景

- 审计：查看谁在什么时候修改了什么配置
- 回滚：需要时可以根据历史恢复配置
- 调试：配置问题排查时查看变更记录

## 常见使用场景

### 场景1：专项练习密码算法

**目标**：在"分类练习"模式下只练习"密码算法"相关的题目

**操作步骤**：
1. 进入"题目范围配置"
2. 找到"📂 分类练习"配置项
3. 选择"范围模式" → "按知识点分类"
4. 勾选"☑ 密码算法"
5. 点击"保存配置"

**效果**：进入"分类练习"模式后，只会出现"密码算法"分类的题目。

---

### 场景2：考试前针对实务综合复习

**目标**：临时限制所有练习模式只使用"实务综合"类别的题目

**操作步骤**：
1. 进入"题目范围配置"
2. 对"随机练习"、"分类练习"、"考试类别"三个配置项：
   - 选择"范围模式" → "按考试类别"
   - 勾选"☑ 密码应用与安全性评估实务综合"
3. 逐一保存配置

**效果**：所有练习模式都会使用实务综合类别的题目。

**恢复**：考试后重置为"全部题目"即可。

---

### 场景3：练习错题集

**目标**：创建一个错题集用于专项练习

**操作步骤**：
1. 查看错题列表，记录题目ID（假设是：1, 2, 3, 5, 8, 13）
2. 进入"题目范围配置"
3. 找到"🎲 随机练习"配置项
4. 选择"范围模式" → "自定义题目"
5. 在"题目ID列表"输入：`1, 2, 3, 5, 8, 13`
6. 点击"保存配置"

**效果**：随机练习模式只会出现这些错题。

**恢复**：练习完成后，重置为"全部题目"。

---

### 场景4：禁用自动复习功能

**目标**：临时禁用SuperMemo自动复习

**操作步骤**：
1. 进入"自动规则"标签页
2. 找到"自动应用SuperMemo算法"
3. 点击开关将其关闭
4. 状态变为"已禁用"

**效果**：系统不再自动计算复习时间。

**恢复**：再次点击开关即可启用。

## 问题排查

### 问题1：配置保存后没有生效

**可能原因**：
1. 浏览器缓存了旧配置
2. 后端服务未运行

**解决方法**：
1. 强制刷新浏览器（Ctrl + Shift + R）
2. 检查"配置历史"确认保存是否成功
3. 查看浏览器控制台是否有错误信息

---

### 问题2：看不到"系统配置"按钮

**可能原因**：
1. 未登录或登录的不是管理员账号

**解决方法**：
1. 使用管理员账号登录（admin / admin123）
2. 刷新页面

---

### 问题3：题目ID输入后没有更新

**可能原因**：
1. 输入了非数字内容
2. 格式不正确

**解决方法**：
1. 确保只输入数字
2. 使用逗号分隔多个ID
3. 不要使用空格（除了ID之间）

**正确格式**：
```
✅ 1, 2, 3, 5, 8
✅ 10,20,30,40
❌ 1, 2, 3, 5, abc   ← 包含非数字
❌ 1-2-3-5      ← 使用了连字符
```

---

### 问题4：分类选择框显示为空

**可能原因**：
1. 数据库中没有分类数据
2. API调用失败

**解决方法**：
