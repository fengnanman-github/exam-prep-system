# 题号显示功能添加报告

**添加时间**: 2026-04-01
**功能**: 在练习页面显示题号

---

## ✅ 实现效果

### 题号显示位置

题号显示在题目的元数据区域，与其他标签（法律法规、技术专业、难度）并列显示：

```
🔢 题号：2599  📚 法律法规大类  🔧 密码算法  📊 中等
```

### 样式特点

- **颜色**: 绿色背景 (#e8f5e9)，深绿色文字 (#2e7d32)
- **图标**: 🔢 (数字键盘图标)
- **字重**: 加粗显示 (font-weight: 600)
- **字号**: 稍大 (0.85rem)，更易识别

---

## 🔧 技术实现

### 1. 模板修改

在 `PracticeMode.vue` 的题目元数据区域添加题号标签：

```vue
<div class="question-meta">
  <span class="meta-tag number-meta">
    🔢 题号：{{ currentQuestion.question_no || currentQuestion.id }}
  </span>
  <span class="meta-tag law-meta">{{ currentQuestion.law_category }}</span>
  <span class="meta-tag tech-meta">{{ currentQuestion.tech_category }}</span>
  <span class="meta-tag difficulty-meta">{{ getDifficultyLabel(currentQuestion.difficulty) }}</span>
</div>
```

**逻辑说明**:
- 优先显示 `question_no` 字段（原始题号）
- 如果 `question_no` 为空，则显示 `id`（数据库ID）
- 使用 `||` 运算符提供后备值

### 2. 样式添加

```css
.meta-tag.number-meta {
  background: #e8f5e9;
  color: #2e7d32;
  font-weight: 600;
  font-size: 0.85rem;
}
```

**颜色选择**:
- 背景色: #e8f5e9（浅绿色）
- 文字色: #2e7d32（深绿色）
- 与其他标签形成区分，但不突兀
