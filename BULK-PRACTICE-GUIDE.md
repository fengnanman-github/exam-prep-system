# 🚀 批量练习功能使用指南

## ✨ 新功能

错题本现已支持**多选批量练习**，让您可以一次性选择多道错题进行针对性练习！

## 🎯 功能特性

### 1. 多选功能
- ✅ **单选**：点击每道题前的复选框
- ✅ **全选**：点击顶部"全选"复选框
- ✅ **取消全选**：再次点击全选复选框
- ✅ **实时计数**：显示"已选 X/总数"

### 2. 批量操作
- 🚀 **批量练习**：一次性练习选中的多道错题
- 🗑️ **批量删除**：一次性删除选中的错题（已掌握）

### 3. 视觉反馈
- 📌 **选中高亮**：选中的错题背景变蓝色
- 🔢 **数量显示**：按钮显示选中数量
- 🚫 **禁用状态**：未选中时按钮禁用

## 📖 使用步骤

### 批量练习流程

1. **打开错题本**
   - 从首页进入"错题本"

2. **选择错题**
   - 方法1：点击"全选"复选框
   - 方法2：逐个勾选要练习的错题

3. **开始批量练习**
   - 点击"🚀 批量练习 (X)"按钮
   - 系统自动跳转到专项练习
   - 自动加载选中的所有题目

4. **按顺序练习**
   - 按题目顺序依次练习
   - 每道题可以添加笔记和收藏
   - 练习完成后查看统计

### 批量删除流程

1. **选择要删除的错题**
   - 勾选已掌握的错题

2. **确认删除**
   - 点击"🗑️ 批量删除 (X)"按钮
   - 确认删除操作

3. **自动刷新**
   - 错题本自动更新
   - 统计数据自动刷新

## 🔧 技术实现

### 状态管理
```javascript
data() {
  return {
    selectedItems: new Set(),  // 使用Set存储选中项
    isAllSelected: false,      // 全选状态
    selectedCount: 0           // 选中计数
  }
}
```

### 批量练习逻辑
```javascript
bulkPractice() {
  // 获取选中题目的question_no
  const questionNos = this.wrongAnswers
    .filter(item => this.selectedItems.has(item.question_id))
    .map(item => item.question_no)
  
  // 触发专项练习，传递逗号分隔的题号
  this.$emit('start-practice', questionNos.join(','))
}
```

### 批量删除逻辑
```javascript
bulkRemove() {
  // 并发删除所有选中错题
  const deletePromises = Array.from(this.selectedItems)
    .map(questionId => axios.delete(`${API_BASE}/wrong-answers/${userId}/${questionId}`))
  
  await Promise.all(deletePromises)
}
```
