# 📝 Markdown笔记功能使用指南

## ✨ 新功能

笔记功能现已支持 **Markdown格式**，让您能够创建更丰富、更有条理的学习笔记！

## 🎯 支持的Markdown语法

### 1. 文本格式

```markdown
**粗体文本** 或 __粗体文本__
*斜体文本* 或 _斜体文本_
~~删除线~~
`行内代码`
```

### 2. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

### 3. 列表

#### 无序列表
```markdown
- 项目1
- 项目2
  - 子项目2.1
  - 子项目2.2
```

#### 有序列表
```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 4. 引用

```markdown
> 这是一段引用文本
>> 嵌套引用
```

### 5. 代码块

````markdown
`行内代码`

​```
代码块
​```
```

### 6. 链接

```markdown
[链接文本](https://example.com)
```

## 💡 实用示例

- **普通密码**：保护普通信息的密码技术

## 解题技巧

## 📚 常见问题

### Q: 如何换行？
A: Markdown中单个换行符会被转换为`<br>`，或者使用两个空格加换行。

### Q: 如何显示特殊字符？
A: 使用反斜杠转义，如 `\*` 显示为 `*`

### Q: 代码块不显示高亮？
A: 当前版本不支持语法高亮，但代码会以等宽字体显示。

### Q: 可以插入图片吗？
A: 当前版本不支持图片，专注于文本内容。
