# 文件名搜索功能使用指南

## 功能概述

专项训练中新增了**按文件名搜索**功能，允许用户通过输入文件名称，快速找到题干中包含该文件名的所有题目进行练习。

## 使用场景

- **按法规学习**：输入"密码法"，练习所有与《密码法》相关的题目
- **按条例学习**：输入"商用密码管理条例"，练习相关题目
- **按文档学习**：输入任何文档名称，找到引用该文档的题目

## 使用步骤

### 1. 进入专项训练

访问系统后，点击导航栏的"专项练习"或首页的"🎯 专项练习"卡片。

### 2. 找到文件名搜索区域

在专项练习页面，向下滚动到"📄 按文件名搜索"部分。

### 3. 输入文件名

在搜索框中输入要查找的文件名，例如：
- `密码法`
- `商用密码管理条例`
- `电子签名法`
- `网络安全法`

### 4. 点击搜索按钮

点击"🔍 搜索"按钮，系统会：
- 在题干中搜索包含该文件名的所有题目
- 显示找到的题目数量
- 展示前20条搜索结果预览

### 5. 添加到练习列表

查看搜索结果后，点击"添加到练习列表"按钮，所有搜索到的题目会自动添加到练习列表中。

### 6. 开始练习

点击"🚀 开始练习"按钮开始答题。

## 功能特性

### ✅ 智能搜索
- **模糊匹配**：支持部分文件名匹配
- **不区分大小写**：搜索"密码法"和"密码法"结果相同
- **题干搜索**：在题目文本中查找文件名

### ✅ 结果预览
- 显示搜索结果总数
- 预览前20条题目
- 每条结果显示题号和题干摘要（前80字符）

### ✅ 批量操作
- 一键添加所有搜索结果到练习列表
- 清空搜索结果重新搜索
- 与手动输入题号功能完全兼容

### ✅ 灵活组合
- 可以先搜索添加一批题目
- 再手动输入补充其他题号
- 支持从错题本、收藏、笔记选择
- 所有来源的题目会合并去重

## 技术实现

### 后端API
- **端点**：`GET /api/questions/search`
- **参数**：
  - `filename`（必填）：要搜索的文件名
  - `limit`（可选）：返回结果数量限制，默认100，最多500
- **返回**：
  ```json
  {
    "success": true,
    "data": [
      {
        "id": 1562,
        "question_no": "48",
        "question_text": "根据《密码法》...",
        "question_type": "单项选择题",
        ...
      }
    ],
    "stats": {
      "total": 2,
      "search_term": "密码法",
      "timestamp": "2026-04-04T15:32:27.163Z"
    }
  }
  ```

### 数据库查询
使用PostgreSQL的`ILIKE`进行不区分大小写的模糊匹配：
```sql
SELECT * FROM questions
WHERE question_text ILIKE '%filename%'
ORDER BY question_no::int
LIMIT limit;
```

## 注意事项

1. **搜索范围**：仅在题干（question_text）字段中搜索，不包括选项和解析
2. **结果限制**：单次搜索最多返回500条结果
3. **性能考虑**：搜索大型数据库可能需要几秒钟时间
4. **URL编码**：axios自动处理中文URL编码，无需手动编码

## 常见问题

### Q: 搜索不到结果怎么办？
A: 请尝试：
1. 缩短搜索关键词（如"密码"而不是"密码法第三章"）
2. 检查是否有错别字
3. 使用更通用的关键词

### Q: 可以搜索多个文件名吗？
A: 当前版本一次只能搜索一个文件名。如需多个，可以：
1. 分别搜索并添加到练习列表
2. 系统会自动去重合并

### Q: 搜索结果太多怎么办？
A: 系统最多返回500条结果。如果需要更精确的搜索，请使用更具体的文件名。
