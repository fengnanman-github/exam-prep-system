## 📋 开发工作流

### 修复Bug
1. `Grep` 搜索错误信息关键词
2. `Read` 只读取相关文件
3. `Edit` 精确修改问题代码
4. `Bash` 快速测试验证

### 添加功能
1. `Glob` 查找类似功能参考
2. `Read` 学习参考实现
3. `Edit` 基于模板创建新文件
4. `Bash` 测试新功能

### 代码审查
1. 使用 `Explore` Agent 全面分析
2. `Grep` 搜索特定问题模式
3. 生成问题列表报告

## 🎨 项目特定信息

### 技术栈
- 前端：Vue3 + Vite
- 后端：Node.js + Express
- 数据库：PostgreSQL (Docker)
- 认证：JWT
- 容器：Docker Compose

### 重要文件
| 文件 | 作用 |
|------|------|
| frontend/src/App.vue | 主应用，路由定义 |
| frontend/src/components/LoginModal.vue | 登录组件 |
| backend/server.js | 后端主服务器 |
| backend/auth/auth-controller.js | 认证控制器 |
| backend/unified-stats-api.js | 统计API |

### 端口配置
| 服务 | 端口 |
|------|------|
| 前端 | 18080 |
| 后端 | 3000 |
| 数据库 | 15432 |
