## 🎯 已完成的优化

### 1. 配置优化 (settings.local.json)

| 优化项 | 效果 |
|--------|------|
| maxSearchResults: 50→20 | 减少返回数据 |
| alwaysUseGlobForFileSearch: true | 使用专门工具 |
| alwaysUseGrepForContentSearch: true | 使用专门工具 |

## 🎯 使用技巧

### 1. 高效提问

**❌ 不好的方式:**
```
"帮我检查一下那个登录功能，好像不太对劲，可能是什么地方出了问题我也不知道"
```

**✅ 好的方式:**
```
"检查LoginModal.vue的登录API调用，返回401错误"
```

### 2. 并行工作

```
"同时检查3个问题：
- Agent1: 检查前端登录组件
- Agent2: 检查后端API
- Agent3: 检查数据库连接"
```

### 3. 使用Skills

```
"重启后端"        → 自动调用restart-backend
"检查状态"        → 自动调用check-backend
"查看用户"        → 自动调用list-users
"docker ps"       → 自动调用docker-skill
```
