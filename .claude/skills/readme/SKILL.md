# README维护Skill

自动维护项目的README.md文件，保持文档与代码同步。

## 功能

- 更新安装说明
- 更新使用指南
- 更新API文档
- 更新配置说明
- 更新贡献指南

## 使用方法

```
readme 更新安装说明
readme 添加API文档
readme 更新项目结构
```

## 自动检测

当以下变更发生时提示更新README：
- 新增依赖包
- API端点变更
- 配置文件变更
- 环境变量变更
- 项目结构变更
