# 移动端快速测试指南

## 访问方式

### 1. 局域网访问（推荐）

#### 查看服务器IP
```bash
# Linux
hostname -I | awk '{print $1}'

# 或使用
ip addr show | grep "inet " | grep -v 127.0.0.1
```

#### 手机访问
1. 确保手机和服务器在同一Wi-Fi网络
2. 打开手机浏览器
3. 访问：`http://[服务器IP]:18080`
4. 例如：`http://192.168.1.100:18080`

### 2. 桌面浏览器测试

#### Chrome DevTools
1. 打开Chrome浏览器
2. 访问 `http://localhost:18080`
3. 按 `F12` 打开开发者工具
4. 按 `Ctrl+Shift+M` 切换到设备模式
5. 选择设备型号（iPhone 12 Pro, Galaxy S20等）

#### 支持的设备预设
- iPhone SE
- iPhone 12 Pro
- iPhone 12 Pro Max
- iPad Pro
- Samsung Galaxy S20
- Pixel 5

## 常见问题

### Q: 导航栏被截断怎么办？
A: 导航栏支持横向滑动，左右滑动即可查看所有导航项。

### Q: 输入框点击时页面放大？
A: 已设置字体为16px，如有问题请清除浏览器缓存重试。

### Q: 按钮点击不响应？
A: 确保点击的是按钮区域，如有问题请刷新页面重试。

### Q: 页面显示异常？
A:
1. 清除浏览器缓存
2. 硬刷新页面（Ctrl+F5）
3. 检查网络连接
4. 尝试其他浏览器

### Q: 如何查看更多信息？
A: 查看 `MOBILE-OPTIMIZATION.md` 文档了解详细优化内容。
