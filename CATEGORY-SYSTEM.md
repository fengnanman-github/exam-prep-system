## 分类特点

### 1. 科学性
- **法律法规维度**：按法律效力层级划分（法律>条例>办法>规划）
- **技术领域维度**：按密码技术体系划分（算法、协议、产品、应用等）
- **交叉组合**：支持"法律+技术"的双维度查询

### 2. 实用性
- **按法备考**：可以针对特定法律法规进行专项练习
- **按技术备考**：可以针对特定技术领域深入学习
- **组合练习**：支持"法律A+技术B"的精准组合练习

### 3. 智能化
- **自动分类**：基于关键词匹配和语义分析
- **权重算法**：完整法律名称权重高于关键词
- **优先级排序**：技术类别按专业程度优先级排序

## API接口

### 获取分类列表

```bash
# 获取两级分类体系（推荐）
GET /api/v2/categories/v2

# 获取法律法规大类
GET /api/v2/categories/law

# 获取技术专业类别
GET /api/v2/categories/tech

# 获取旧版分类（兼容）
GET /api/v2/categories
```

### 按分类获取题目

```bash
# 按法律法规大类查询
GET /api/v2/questions/category/{category}?type=law

# 按技术专业类别查询
GET /api/v2/questions/category/{category}?type=tech

# 按两级组合查询
GET /api/v2/questions/two-level?law_category={law}&tech_category={tech}

# 练习模式（支持两级筛选）
GET /api/v2/questions/practice?law_category={law}&tech_category={tech}&difficulty={level}
```

- 先学习《密码法》等基础法律

- 使用组合查询精准定位知识点

### 3. 模拟考试
- 选择多个法律法规大类组合
- 覆盖主要技术专业类别
- 模拟真实考试难度分布

## 技术实现

分类算法基于：
1. 关键词匹配（完整法律名称 > 关键词）
2. 上下文分析（题目+选项+知识点）
3. 权重计算（多维度评分）
4. 优先级排序（技术类别专业度）

所有题目均已自动分类，支持后续人工校准和优化。
