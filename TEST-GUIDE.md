## 📋 测试步骤

### 准备工作

1. **打开浏览器控制台**（F12 → Console标签页）
   - 用于查看日志输出
   - 帮助诊断问题

2. **访问系统**
   - URL: http://localhost:18080
   - 确保已登录

---

### 测试流程

#### 第1步：进入文档练习页面

1. 点击顶部导航的 **"文档"** 按钮
2. 等待页面加载完成
3. 在控制台应该看到：
   ```
   DocumentReview: 统一API状态 true/false
   ```

#### 第2步：选择筛选条件

1. **文档类别**：选择"技术标准"（应该显示10个文档）
2. **优先级筛选**：选择"核心必考"
3. 找到 **"SM2椭圆曲线公钥密钥算法"** 文档
4. 确认显示：
   - 题目数：185
   - 已练：0
   - 正确率：0%

#### 第3步：开始练习

1. 点击 **"开始练习"** 按钮
2. 在弹出的对话框中：
   - 练习模式：选择"随机顺序"
   - 题目数：改为 **2**
   - 点击 **"开始练习"**

#### 第4步：完成练习

1. **做第1题**：
   - 选择答案
   - 点击"提交答案"
   - 查看结果（正确/错误）
   - 控制台应该显示：
     ```
     📝 提交文档练习答案: {document_name: "SM2椭圆曲线公钥密钥算法", ...}
     ✅ 练习历史记录成功: {...}
     ```

2. **做第2题**：
   - 重复上述步骤
   - 控制台应该再次显示提交成功的日志

3. **完成练习**：
   - 完成第2题后会自动显示总结
   - 点击"确定"或等待自动返回

#### 第5步：验证数据更新

练习完成后应该自动返回文档复习页面，此时：

1. **检查控制台日志**：
   ```
   DocumentReview: 组件被激活，强制刷新数据
   🔄 开始加载文档数据, forceReload: true
   📊 收到统计数据: {...}
   ```

2. **再次找到"SM2椭圆曲线公钥密钥算法"**：
   - 已练：应该显示 **2** ✅
   - 正确率：应该显示 **50%** 或 **100%**（取决于答对几题）✅

## 🐛 如果问题仍然存在

### 检查清单

#### 1. 检查控制台日志

**应该看到的日志**：
```javascript
// 提交答案时
📝 提交文档练习答案: {
  document_name: "SM2椭圆曲线公钥密钥算法",
  is_correct: true/false,
  question_id: xxx
}

✅ 练习历史记录成功: {...}

// 返回页面时
DocumentReview: 组件被激活，强制刷新数据
🔄 开始加载文档数据, forceReload: true
📊 收到统计数据: {...}
```

**如果看到错误**：
```javascript
❌ 记录练习历史失败: {...}
```

#### 2. 检查网络请求

1. 打开DevTools的 **Network** 标签
2. 筛选 **XHR** 请求
3. 练习完成后应该看到：

**POST /api/v2/practice/history**
- Status: 200 ✅
- Response: 包含提交的数据

**GET /api/v2/stats/user/{userId}**
- Status: 200 ✅
- Response: 包含最新的统计数据

#### 3. 常见问题排查

**问题1：返回到首页而不是文档页面**
- 原因：事件处理有问题
- 解决：检查PracticeMode.vue中的showCompletionSummary方法

**问题2：数据没有更新**
- 原因：API请求失败或缓存问题
- 解决：查看Network标签中的请求和响应

**问题3：控制台没有日志**
- 原因：浏览器缓存了旧版本
- 解决：硬刷新（Ctrl+Shift+R）

如果自动刷新不工作，可以手动刷新：
- 点击页面顶部的"文档"导航按钮

### 2. 检查数据库

```bash
# 查看练习历史
docker exec exam-prep-system-package-20260330-db-1 psql -U exam_user -d exam_db -c "
SELECT question_id, is_correct, practiced_at
FROM practice_history
WHERE user_id = 'your_user_id'
ORDER BY practiced_at DESC
LIMIT 5;
"
```

### 3. 查看后端日志

```bash
docker logs exam-prep-system-package-20260330-backend-1 --tail 50
```
